# 社团管理系统 java1043

## 项目概述

社团管理系统是一个基于`SpringBoot`和`Vue`的前后端分离项目。项目致力于为校园社团提供一套完整的信息管理解决方案，以提高社团管理效率和活动组织水平。

## 技术栈

- **前端：** Vue
- **后端：** SpringBoot、MyBatis-Plus
- **数据库：** MySQL

## 功能模块

- 用户注册
- 用户登录
- 修改密码
- 用户信息修改
- 社团类型管理
- 入团申请管理
- 活动信息管理
- 费用管理

## 系统角色

本项目主要涉及以下角色：
- 用户：进行注册、登录、信息修改等操作。
- 社团管理员：负责社团类型管理、申请管理、活动发布及费用管理等。

## 运行环境

- **前端：** 现代浏览器
- **后端：** JDK 1.8 或以上版本，运行在兼容 SpringBoot 的服务器上
- **数据库：** MySQL 5.7 或以上版本

## 部署步骤

1. 准备 MySQL 数据库环境，导入 SQL 文件创建所需数据表。
2. 部署后端服务，配置好数据库连接。
3. 部署前端应用，确保与后端服务网络连接正常。
